Executes a POST request to upload a media item.

Properties
Name Type Description

Properties
Name Type Description
content string The base64 representation of the media item.
contentType string The content type of the item. E.g. image/jpeg.
name string The name of the media item.
payload string Holds the JSON metadata about the image. E.g. Title, Urlname etc.
library string The name of the navigation property.
link string The URL of the related library item.
successCb function The success callback function.
failureCb function The error callback function.

Example

    urlName: "images"
});

    data: {
        content: "base 64 representation of the file",
        contentType: "image/jpeg",
        name: "test.jpg",
        payload: {
            Title: "Test",
            UrlName: "test"
        }
    },
    library: "Parent",
    link: "http://localhost/api/default/albums(8818c109-3515-4a74-bf3f-d5f1d5e201a3)",
    successCb: successCb,
    failureCb: failureCb
});
